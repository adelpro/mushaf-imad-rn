**Mushaf Imad RN** is a high-performance, open-source Quran application built with React Native. It utilizes a sophisticated SVG-over-image mapping system to provide a premium, paper-like reading experience with interactive verse highlighting and synchronized audio playback.

---

## ‚ú® Key Features

- **Standard 604-Page Mushaf**: High-quality Quranic pages with authentic layout.
- **Dynamic Interaction**: 
  - Precise verse highlighting using SVG layers.
  - Interactive verse markers (Fasel) with Arabic numbering.
  - Surah title bars rendered dynamically.
- **Synchronized Audio**: Built-in audio player with verse-by-verse synchronization using timing metadata.
- **High Performance**: Optimized using **Realm DB** for local data persistence and efficient asset management.
- **Modern UI/UX**: Clean, responsive interface designed for spiritual focus.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18+)
- Yarn or NPM
- Expo Go (for development) or a Native Environment (Android/iOS)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/adelpro/mushaf-imad-rn.git
   cd mushaf-imad-rn
   ```

2. **Install dependencies**:
   ```bash
   yarn install
   # or
   npm install
   ```

3. **Start the development server**:
   ```bash
   yarn start
   ```

4. **Run on a device**:
   - For Android: `yarn android`
   - For iOS: `yarn ios`
